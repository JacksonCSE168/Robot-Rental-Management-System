# Robot-Rental-Management-System
This project is a database-driven Robot Rental Management System built using Java and SQLite.

The system supports core CRUD operations (Create, Read, Update, Delete) for managing customers, robots, rentals, and related entities. It also implements real-world rental workflows, including renting robots, returning equipment, scheduling deliveries, and pickups.

In addition, the system provides analytical reports such as most popular robots, manufacturers, and rental activity insights. SQL queries are optimized using joins and aggregation functions, and the application features a command-line interface for user interaction.

This project demonstrates practical database design (relational schema, foreign keys), SQL querying, and backend logic integration in Java.
